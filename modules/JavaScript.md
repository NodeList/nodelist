# JavaScript
JavaScript有以下三部分组成：
> - ECMAScript(核心)：ECMAScript提供核心语言功能 
> - 文档对象模型(DOM)：DOM提供访问和操作网页内容的方法和接口
> - 浏览器对象模型(BOM)：BOM提供与浏览器交互的方法和接口
